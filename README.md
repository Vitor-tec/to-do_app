# 📋 Projeto: Lista de Tarefas (To-Do List)

## Descrição do Projeto

Este projeto consiste no desenvolvimento de uma **Lista de Tarefas (To-Do List)** simples, funcional e organizada, criada por um desenvolvedor iniciante com o objetivo de demonstrar conhecimentos básicos em desenvolvimento web utilizando **HTML, CSS e JavaScript**.

A aplicação permite ao usuário adicionar tarefas, definir prioridades, visualizar as tarefas em uma lista dinâmica e marcar tarefas como concluídas. O sistema também utiliza cores diferentes para cada nível de prioridade, facilitando a identificação visual e melhorando a usabilidade.

O projeto foi desenvolvido seguindo boas práticas de organização de código, separação por arquivos e princípios básicos de acessibilidade e usabilidade, estando pronto para testes, avaliação de desempenho e apresentação profissional.

---

## Objetivo

O objetivo principal deste projeto é criar uma aplicação web simples e funcional que permita:

* Registrar tarefas
* Definir prioridade para cada tarefa
* Exibir tarefas dinamicamente
* Marcar tarefas como concluídas
* Demonstrar conhecimentos básicos em desenvolvimento front-end
* Servir como base para avaliação técnica e apresentação profissional

---

## Funcionalidades

* Campo de entrada para descrição da tarefa
* Seletor de prioridade (Alta, Média, Baixa e Sem prioridade)
* Lista dinâmica de tarefas
* Botão para marcar tarefa como concluída
* Estilização condicional com cores por prioridade
* Interface centralizada na página
* Validação simples de entrada

## Como Testar o Sistema

Para realizar testes de funcionamento:

1. Digite uma tarefa no campo de entrada.
2. Selecione a prioridade desejada.
3. Clique no botão **Adicionar Tarefa**.
4. Verifique se a tarefa aparece na lista.
5. Clique no botão **Concluir** para marcar a tarefa como finalizada.

Critérios de teste:

* Adição de tarefas
* Mudança de cor conforme prioridade
* Marcação de tarefa como concluída
* Funcionamento sem erros
